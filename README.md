> To connect to a server.
1. Open Multiplayer
2. Click Direct Connect
3. Click Connect to Server
4. Paste the Server Link

> To Host. `Version 1.8 Only`
1. Click Multiplayer Setup
2. Follow Steps

> To join a host. `Version 1.8 Only`
1. Open Multiplayer
2. Click Direct Connect
3. Click Join Shared World
4. Enter Join Code
